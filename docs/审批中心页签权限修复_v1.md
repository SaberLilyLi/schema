# 需求变更记录：审批中心页签权限修复

## 问题

普通用户在审批中心中可能看到“待我审批”页签，不符合权限设计。

## 修复

文件：`frontend/src/views/approval/ApprovalCenterView.vue`

- 将 `canApprove` / `canSubmit` 改为响应式 `computed`
- 无 `approval:approve` 时强制不进入 “待我审批” 页签
- 页签选项按实时权限计算：
  - `approval:approve` -> 显示“待我审批”
  - `kb:submit` -> 显示“我发起的”

## 权限文档同步

文件：`docs/ROLE_ACCOUNTS.md`

- 新增“审批中心页签权限”说明：
  - 待我审批：`approval:approve`
  - 我发起的：`kb:submit`

## 验证结果

- 前端构建通过
- 普通用户不再显示“待我审批”页签
