# 需求变更记录：审批历史与搜索增强

- 审批中心新增“历史审批记录”

- 审批中心补齐查询搜索项

## 后端变更

文件：`backend/src/routes/articles.js`

- `POST /api/articles/approval/pending`
  - 新增可选筛选：`q`、`domain`
- `POST /api/articles/approval/initiated`
  - 新增可选筛选：`q`、`domain`、`workflowState`
- 新增 `POST /api/articles/approval/history`
  - 权限：`approval:approve`
  - 支持筛选：`q`、`domain`、`workflowState`、`approver`
  - 返回字段新增：`approverName`
- 驳回动作补充记录审批人信息（写入 `approvedBy` / `approvedAt`），确保历史记录可展示审批人
